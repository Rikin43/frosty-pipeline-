## 🍦 FrostyPipeline — Project Overview

**FrostyPipeline** is a DevOps project that demonstrates how to automatically deploy a static website to **AWS S3** using **GitHub Actions** CI/CD pipeline — triggered on every `git push` to the main branch.

The project includes two static websites built with pure **HTML & CSS** — a DevOps-themed landing page and a fully designed Ice Cream shop website — both hosted live on AWS S3 as static sites.

It covers real-world DevOps practices like **IAM credential management**, **GitHub Secrets**, **S3 bucket policies**, and **zero-downtime automated deployments** — making it a perfect beginner-to-intermediate portfolio project.
